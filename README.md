# Hong Kong maimai DX / IIDX Arcade Map

Static single-page map for Hong Kong arcades with maimai DX and/or beatmania IIDX.

Generated on 2026-06-16 from BEMANI CN, Zenius-I-vanisher, MTR fare data, and public border bus fare references.

## Deploy

This folder is ready for static hosting. Use `index.html` as the site entry point.

Suitable hosts:

- GitHub Pages
- Cloudflare Pages
- Netlify
- Vercel static project

## Update

Refresh the source data, rebuild `index.html`, then redeploy this folder.
